# 🏥 Sistema de Gestão de Farmácias

Sistema desenvolvido em C para gerenciamento de farmácias, com funcionalidades de cadastro de funcionários, clientes, produtos e vendas.

## 📋 Funcionalidades

- 🔐 Sistema de login com autenticação
- 👨‍⚕️ Gestão de Funcionários (CRUD)
- 💊 Gestão de Produtos (CRUD)
- 👤 Gestão de Clientes (CRUD)
- 💰 Gestão de Vendas (CRUD)
- 🎨 Interface no terminal com efeitos visuais

## 📋 Requisitos do Sistema

### Capacidade Máxima
- 👨‍💼 **Funcionários:** 30 registros
- 💊 **Produtos:** 100 registros
- 👥 **Clientes:** Ilimitado
- 💰 **Vendas:** Ilimitado (histórico completo)

### Requisitos Técnicos
- Sistema Operacional: Windows (com suporte a `windows.h`)
- Compilador: GCC (com suporte a C99 ou superior)
- Memória: 512 MB RAM mínimo
- Espaço em disco: 50 MB para arquivos de dados
